Block Carousel

Reto Stauffer

2022-08-18

Block carousel

Rotorweb provides a shortcode to create in-line block image slideshows (or block carousels). The shortcode {{< blockcarousel >}} takes up a series of images to be displayed.

{{< blockcarousel [image1] [image2] ... [options] >}}

Example

The following two shortcodes create the carousels displayed below:

{{< blockcarousel ../header-images/unsplash-1.jpg ../header-images/unsplash-2.jpg >}}
{{< blockcarousel ../header-images/unsplash-3.jpg ../header-images/unsplash-4.jpg ../header-images/unsplash-5.jpg >}}

… creates the carousel shown below. The images are cycled automatically but can be navigated trough using the two buttons shown on the right/left of the image or via swiping.

Additional arguments

A series of additional options are available which can be specified via key=value pairs when setting up the inline block carousel. Besides id all options are directly forwarded to the bootstrap carouse, for more details see bootstrap 5.0 carousel options.

Name Type Default Description
id string random ID of the HTML element. By default, a random ID is generated.
interval string 5000 Amount of time (milliseconds) to delay between automatically cycling an item.
keyboard boolean false Whether the carousel should react to keyboard events.
pause string or boolean hover Default is to pause cycling when the mouse cursor is over the carousel.
ride string or boolean carousel By default, the carousel is on autoplay.
wrap boolean true Continuous cycling by default, can be set to false for hard stops.
touch boolean true Can be set to false to disable swiping on touchscreen devices.

Example block carousel with all options explicitly specified. Besides id and interval all options are set to default.

{{< blockcarousel ../header-images/unsplash-1.jpg ../header-images/unsplash-2.jpg id="block_carousel1" interval=1000 keyboard=true pause=hover ride=carousel wrap=true touch=true >}}
