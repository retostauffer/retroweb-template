Getting started

To get started we need to clone the template repository as well as cloning the quarto extension which contains the Retroweb extension.

git clone https://github.com/retostauffer/retroweb-template.git mywebsite
cd mywebsite
git submodule add https://github.com/retostauffer/retroweb-ext.git _extensions/retroweb

When cloned, the extension can be installed and the page rendered.

quarto install extension .
quarto render

The website will be rendered into docs/.

Block-carousel shortcode

{{< blockcarousel [image1] [image2] ... id=[elementId] >}} allows to create block carousels within the content of the page. Currently no additional support for options (TODO(R)).
