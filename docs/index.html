Getting started

To get started we need to clone the template repository as well as cloning the quarto extension which contains the Rotorweb extension.

git clone https://github.com/retostauffer/rotorweb-template.git mywebsite
cd mywebsite
git submodule add https://github.com/retostauffer/rotorweb-ext.git _extensions/rotorweb

When cloned, the extension can be installed and the page rendered.

quarto install extension .
quarto render

The website will be rendered into docs/.
